# **App Name**: Quantum Leap Academy

## Core Features:

- Course Catalog: Display all available courses, each with a detailed description, instructors, learning path, and student reviews.
- Personalized Learning Path: Suggest an optimal learning path depending on current job profile, and desired career trajectory.  Will act as a tool that suggests relevant resources (internal courses and external links) based on student progress, and generate a high-level roadmap to show the student a career vision.
- AI-Enhanced Resume Builder: Enable students to create resumes optimized for specific roles and industries. Will act as a tool that offers smart suggestions based on skills, experience, and target job descriptions.
- Live Class Booking: Allow users to book and attend live classes.
- Student Success Stories: Display success stories and testimonials from past students.
- Instructor Profiles: Showcase instructor profiles with their expertise and background.
- Admin Panel: An admin panel for managing courses, users, and content.

## Style Guidelines:

- Primary color: A professional, optimistic, but non-flashy light blue (#5DADE2).
- Background color: Very light blue/off-white (#F0F8FF).
- Accent color: Muted lavender (#A393C7) to provide a subtle contrast, giving some pop to key actionable elements.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern appearance. It will provide a clean and professional look to the website, suitable for both headings and body text.
- Use modern and professional icons to represent course categories and features.
- A clean, grid-based layout with ample whitespace for easy readability.
- Subtle animations and transitions for a smooth user experience.
